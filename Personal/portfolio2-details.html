Pet Advisor

Project information

  • Category: Fully Responsive Website
  • Client: Passion Project (Prior: CodeX Web Challenge)
  • Project Duration: March 2022 - April 2022,
    Mar 2024 - Current
  • Project URL:
    https://pet-advisor.github.io/pets/index.html

According to Gfk, a global research firm, more than 56% of the world's population has atleast one pet in their household. Despite this staggering number, pet owners still find difficulties in terms of pet management. This website aims to tackle a few of these problems. The website has many features like a vaccine diary, a name picker, resources, and maps to help current and future pet-owners.
